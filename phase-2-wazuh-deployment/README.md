# Phase 2 — Wazuh Deployment

## Objective
The goal of this phase is to deploy Wazuh as the primary SIEM and EDR solution 
of the SOC lab. This includes installing the Wazuh Manager on a dedicated Ubuntu 
VM, deploying the Wazuh Agent on the Windows 11 Pro target machine, and verifying 
that security events are correctly collected and visible in the Wazuh Dashboard. 
Wazuh will act as the core detection engine of the lab, receiving and processing 
all security events generated by the monitored endpoints.

## Components
- **Wazuh Manager** — Installed on Ubuntu VM (192.168.10.30). Receives and 
processes alerts from all agents.
- **Wazuh Agent** — Installed on Windows 11 Pro (192.168.10.20). Collects 
security events and forwards them to the Manager.
- **Wazuh Dashboard** — Web interface accessible at https://192.168.10.30 
for alert visualization and management.
